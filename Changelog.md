# Fortnite Optimizer Changelog

**Fortnite-Optimizer V1.0**

6/24/24 <3

*What it did?*
- Downloads Roblox Optimizer Tools folder that contains GameUserSettings presets.
- Disable Fullscreen Optimizations.
- Set the Graphics Preference to high.
- Set Fortnite to High Priority.
- Auto Install and auto imports GameUserSettings presets of your choosing!
- Guides you to  Additional Command Line Arguments
