# Additional Command Line Arguments

# #1 
Open ```Epic Games Launcher```, then click your ```Profile icon```, then click ```Settings``` and ```scroll all the way down```.

# #2
Expand  ```Fortnite```, Check  ```Additional Command Line Arguments ```.

# #3
Paste the following ↓
```
-limitclientticks -nosplash -useallavailablecores
```
