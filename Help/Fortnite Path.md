# Fortnite Path
If Fortnite isn't in the deafult path or ```C:\Program Files\Epic Games\Fortnite\FortniteGame\Binaries\Win64\FortniteClient-Win64-Shipping.exe``` you get an error message. 

# Manually Input Path
Selecting Manually Input Path allows you to attach the correct path of your Fortnite. 

1. Select ```Manually Input Path```!
2. Enter your Fortnite Path. (Must contain the full path to the exe)

> [!NOTE]
> To find the full Fortnite Path, open Fortnite -> Task Manager -> Right-Click Fortnite -> Open File Location, it should then bring you to the ```FortniteClient-Win64-Shipping.exe```.
> Simply right-click it, and copy as file location and paste into the optimizer!
